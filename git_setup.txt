# Après avoir installé Git, exécutez ces commandes dans l'ordre :

# 1. Initialiser le dépôt
git init

# 2. Ajouter tous les fichiers
git add .

# 3. Faire le premier commit
git commit -m "Initial commit : App Push-ups Challenge 365 jours"

# Pour connecter à GitHub plus tard :
# 1. Créez un nouveau dépôt sur GitHub
# 2. Exécutez ces commandes en remplaçant [URL] par l'URL de votre dépôt :
git remote add origin [URL]
git branch -M main
git push -u origin main
